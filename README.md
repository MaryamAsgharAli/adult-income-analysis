# Adult Income Analysis

Exploratory data analysis and storytelling project on the Adult Income dataset using Python and Tableau.

## Project Overview

This project explores which factors are associated with earning more than $50K per year.

The analysis focuses on:
- income distribution
- education level and income relationship
- working hours and income relationship

## Main Finding

Higher levels of education are strongly associated with a higher likelihood of earning more than $50K per year in the dataset.
